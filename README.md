## 介绍
HNIE易班打卡项目。

### 说明

伴随着疫情的到来，学校为了解同学健康状况，都规定在特定的时间进行打卡，本项目旨在让大家从每天打卡的压力中解放出来，全身心地投入到社会主义建设之中去。

若去过中高风险地区应主动上报。

### 安装教程

1. 打开终端 输入pip install -r requirements.txt  若提示无pip。去下载get-pip.py 运行python get-pip.py
2. 安装成功后，可使用

### 使用说明

1. 在 name.py 中按格式添加学号和密码
2. 运行 main.py
